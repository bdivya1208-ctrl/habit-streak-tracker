# 🔥 Habit Streak Tracker

A modern habit tracking web application that helps users build consistency by tracking daily habits over a 30-day calendar.

## ✨ Features

- Add and delete habits
- 30-day interactive habit calendar
- Current streak tracker
- Longest streak tracker
- Progress bar
- Achievement badges
- Dashboard statistics
- Random motivational quotes
- Saves data using localStorage
- Responsive Glassmorphism UI

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- LocalStorage

## 📂 Project Structure

habit-streak-tracker/

├── index.html

├── style.css

├── script.js

└── README.md

## 🚀 How to Run

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Add your habits.
4. Track your daily progress.
5. Data is automatically saved in your browser.
